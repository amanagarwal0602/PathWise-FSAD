================================================================================

================================================================================

1. Open new browser/tab

   - Password: general123
3. You will see the Career Coordinator Dashboard
4. In "Inbox & Triage" or student list:
   - Find the test student
   - View their assessment results
   - Read their chat messages
5. Chat with the student:

   - Discuss their career interests
6. Assign to Specialized Career Mentor:

   - Select the student

   - Click "Assign Mentor"

================================================================================
                    TROUBLESHOOTING
================================================================================

Student not seeing verification update?
- Click "Refresh Status" button
- Or wait for auto-refresh (every 5 seconds)
- Or log out and log back in

Career Mentor not seeing verification update?
- Same as above - wait for auto-refresh or re-login
- Make sure you registered as "Career Mentor" not "Student"

Wrong evaluator type?
- Evaluator1 & Evaluator2 → Only see STUDENTS pending verification
- Evaluator3 & Evaluator4 → Only see CAREER MENTORS pending verification

Data not clearing between tests?
- Clear localStorage in browser DevTools
- Application tab → Local Storage → Delete pathwiseData

Chats showing old messages?
- Each user has separate chat history
- Demo student (sample@gmail.com) resets on every login

================================================================================
                    SECURITY FEATURES IMPLEMENTED
================================================================================

PathWise includes several security enhancements to protect user data:

PASSWORD SECURITY:
1. SHA-256 Password Hashing
   - All new user passwords are hashed with SHA-256
   - Unique salt generated for each password
   - Original passwords never stored
   - Legacy accounts still work with plaintext (for demo)

2. Strong Password Requirements
   - Minimum 8 characters
   - Must include uppercase letter (A-Z)
   - Must include lowercase letter (a-z)
   - Must include number (0-9)
   - Special characters recommended (!@#$%^&*)
   - Common passwords rejected

3. Password Strength Indicator
   - Real-time strength feedback during registration
   - Visual progress bar (Very Weak → Very Strong)
   - Specific improvement suggestions shown

LOGIN SECURITY:
1. Rate Limiting / Brute Force Protection
   - Maximum 5 failed attempts per 15 minutes
   - Account locks for 5 minutes after limit exceeded
   - Countdown timer shows remaining lockout time
   - Prevents password guessing attacks

2. Secure Session Management
   - Session tokens generated on login
   - 24-hour session expiry
   - Secure logout clears all session data

INPUT VALIDATION:
1. Email Validation
   - Format verification
   - Sanitization to prevent injection

2. Phone Number Validation
   - 10-15 digit requirement
   - Invalid characters stripped

3. XSS Protection
   - All user inputs sanitized
   - HTML entities escaped
   - Prevents script injection attacks

VERIFICATION WORKFLOW:
1. Multi-level Access Control
   - Students verified by Evaluator 1-2
   - Mentors verified by Evaluator 3-4
   - Prevents unauthorized platform access

2. Status Tracking
   - Pending, Verified, Rejected states
   - Rejection reasons logged
   - Verification timestamps recorded
